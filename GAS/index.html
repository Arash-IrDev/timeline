You can also access timeline functions from the "Timeline Tool" menu in your Google Sheet:

  • Show Timeline - Opens the timeline sidebar
  • Setup Sample Data - Adds sample timeline data to your sheet
  • Test Timeline - Tests the timeline setup and returns status

How to use:

  1. Click "Setup Sample Data" to add sample timeline data to your sheet
  2. Click "Show Timeline" to display the timeline in a sidebar
  3. Modify the data in your sheet and refresh the timeline
  4. Use "Test Timeline" to verify everything is working
Status: Ready

Data Format:

Your sheet should have the following columns:

  • Column A: Task/Event name
  • Column B: Start date (YYYY-MM-DD format)
  • Column C: End date (optional, YYYY-MM-DD format)

Leave Column C empty for single-day events.
